# Good First Issues

These are intentionally scoped for new contributors.

1. Add a compact empty-state illustration for the workspace list.
2. Add a copy button to every generated install command in missing-tool panels.
3. Add unit tests for `.env` parsing edge cases.
4. Add fixture coverage for `setup.py` dependency extraction.
5. Add a search filter to the package hygiene root-package modal.
6. Add a visible progress message while syncing project dependencies.
7. Add a retry button to failed background-job result panels.
8. Add a command palette item for opening Cache Hygiene.
9. Add a small badge showing whether an environment was detected as `pip` or `uv` in comparison modals.
10. Add an option to sort package list by size, name, or version.
