# Project tools

Studio includes project-level tools next to package and diagnostics workflows.

## Configuration

- Structured `.env` editor.
- Awareness of `.env.example` and `.env.template`.
- Secret-like value masking.
- Raw edit mode.
- Read-only `pyvenv.cfg` viewer.

## Developer tools

- Saved automation scripts.
- Quick runners for pytest, ruff, black and mypy.
- Open activated terminal.
- Generate VS Code settings.
- VS Code Interpreter Doctor.
- Register Jupyter kernel.
- Generate Docker files.
- Run Docker build/run in a terminal.
- Install pre-commit hooks.
